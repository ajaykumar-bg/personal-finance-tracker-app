# 💰 Personal Finance Tracker App

A production-ready Personal Finance Tracker built with **Expo**, **React Native**, **Redux Toolkit**, and **React Native Paper**. Track your expenses, manage budgets, and visualize spending trends with beautiful charts.

## 🚀 Quick Start

### Installation
```bash
npm install
```

### Run the App
```bash
# Development mode
npm start

# iOS simulator
npm run ios

# Android emulator
npm run android

# Web browser
npm run web
```

## ✨ Features

- ✅ **Track Transactions** - Add, edit, and delete income and expenses
- ✅ **Smart Filtering** - Filter by category, type, or date range
- ✅ **Visual Analytics** - Monthly spending trends and category distribution charts
- ✅ **Budget Management** - Set and monitor monthly budgets by category
- ✅ **Offline Storage** - All data persists locally with AsyncStorage
